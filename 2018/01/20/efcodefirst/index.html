EF Core 介绍

Entity Framework(EF) Core 是Entity Framework 的一个轻量级和可扩展的版本。EF Core 是一个ORM(对象关系映射)，它使.NET开发人员可以直接使用.NET对象来操作数据库，消除了大部分的数据访问代码，开发通常只需要编写对象即可。EF Core支持多种数据库引擎，例如:

  • Microsft SQL Server
  • SQLite
  • Npgsql
  • MySQL
  • Micosoft SQL Server Compact Edition
  • IBM Data Servers
  • InMemory
