在 macOS 中安装SF Mono字体

2021-02-28 1 分鐘閱讀 #macOS#Font

在 macOS 中，SF Mono 字体不是系统自带的字体，而是包含在Terminal.app中的字体，因为如果需要在其他的 App 使用到这个字体的话，就需要将这个SF Mono字体安装在系统当中。
